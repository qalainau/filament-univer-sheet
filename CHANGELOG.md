# Changelog

All notable changes to `filament-univer-sheet` will be documented in this file.

## 1.0.0 - 2026-04-18

- Initial release
- `SpreadsheetField` — full spreadsheet editor for Filament forms
- `SpreadsheetEntry` — read-only spreadsheet display for infolists
- `SpreadsheetColumn` — compact table column preview
- `UniverSheetPlugin` — panel-level configuration
- English and Japanese translations
- Univer Sheet integration via esbuild IIFE bundle
